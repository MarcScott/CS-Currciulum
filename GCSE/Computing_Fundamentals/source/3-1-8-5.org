Secondary storage in Computer Systems, describes anything that stores data more permanently.
Secondary storage, is therefore distinct from RAM and Cache memory, in that it is non-volatile.

Generally, secondary storage media can be classified into one of three categories.
- Magnetic Storage
- Optical Storage
- Solid State Storage
All three are capable of storing data for long periods of time.
