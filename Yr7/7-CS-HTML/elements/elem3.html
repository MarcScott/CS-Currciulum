Create a List - description list

    A description list is a list of terms or names, with a description of each term or name. This can also be used to define our list of hyperlinks.
    1. The <dl>tag defines a description list.
    2. The <dl> tag is used together with <dt> (defines a term) and <dd> (describes a term):
      A description list code
      <dl>
       <dt> Milk </dt>
       <dd> a white liquid<dd>
       <dt> Butter </dt>
       <dd> a diary product<dd>
      </dl>       		What you see in a browser
      Milk
      a white liquid
      Butter
      a diary product
    3. As you see we can use a description list to organise items into categories. We can modify the above code to include more descriptions for each item:
      4. A description list code
      <dl>
       <dt> Milk </dt>
       <dd> a white liquid<dd>
       <dt> Butter </dt>
       <dd> a diary product<dd>
       <dd> used for cooking<dd>
       <dd> produced by churning milk<dd>
      </dl>       		What you see in a browser
      Milk
      a white liquid
      Butter
      a diary product
      used for cooking
      produced by churning milk
