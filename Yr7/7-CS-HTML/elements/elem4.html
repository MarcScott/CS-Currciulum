Create a List - a description list of hyperlinks

    In the leftbox of our design, there is a list of hyperlinks. We can add some descriptions under each hyperlink to give more information as what each link is about.
    1. The <dt> (defines a term) tag will be used to define a hyperlink and the <dd> (describes a term) will be the description for that hyperlink:
      A description list code
      <dl>
       <dt><a href="mySecondPage.html">Photo Gallery</a> </dt>
       <dd> - A collection of photos of Jack from 4 weeks to present><dd>
      </dl>       		What you see in a browser
      Photo Gallery
      - A collection of photos of Jack from 4 weeks to present
